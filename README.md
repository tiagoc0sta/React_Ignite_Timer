
# Countdown Timer  - ReactJS   

ReactJS - Implementing a Countdown on-line timer.

 
Technologies used:

- ReactJS
- Typescript
- Phosphor-react 
- Styled components 
- React router doom 
- React hook form 
- Zod 
- Context provider 
- Reducer 
- Immer JS
